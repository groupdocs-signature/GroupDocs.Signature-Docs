# GroupDocs.Signature - Documentation Source Files

This repository contains Markdown files used to generate [GroupDocs.Signature](https://docs.groupdocs.com/signature/) static documentation.

## About GroupDocs.Signature

[GroupDocs.Signature](https://products.groupdocs.com/signature/) is an electronic signature SDK that enables you to sign documents with text, image, barcode, QR code, and digital signatures across 90+ file formats.

Available for .NET, Java, Node.js, and Python.

## Repository Purpose

The files in this repository are used to:

- Build and maintain the official [documentation](https://docs.groupdocs.com/signature/).
- Ensure documentation is always up-to-date with the latest product release.
- Provide a consistent structure for automated builds and publication.

## How to Use

This repository is not meant for direct use, as it contains source files for generating static content with Hugo.
